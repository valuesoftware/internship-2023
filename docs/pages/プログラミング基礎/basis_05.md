## 5. 変数を知ろう
   
- 26行辺り: ``{/* 5 */}``を確認してください
  
        // JavaScriptの基本の変数宣言
        let num = number + 1


!!! Note
	変数とは、数値や文字などの値を入れられる領域(箱)のことです。

	プログラミングでは、**「値を入れる箱」**を用いてコードを書くことで、何度でも数値を入れたり、入っている数値を見たり、様々な計算などに利用することができるようになります。

		// 構文
		let 変数名 = 初期値;
