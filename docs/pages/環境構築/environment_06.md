#　<i class="fa fa-arrow-circle-right" aria-hidden="true"></i> 環境構築

## 6. リポジトリをclone

!!! Warning
	実行ディレクトリに注意

    以下のディレクトリで行います

    - Windows
        - `c:\intern\app\`

    - MacBook
        - `/Users/{ユーザ名}/intern/app`

GitHubからリポジトリをクローン
   
```
$ git clone https://github.com/valuesoftware/star-search-app.git
```
